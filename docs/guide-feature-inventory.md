# 권한별 페이지 기능 인벤토리

> DEV-1795 설명서 페이지 전면 개편의 원천 자료.
> 코드 분석으로 추출한 강사/학생 권한별 전체 사용자 기능 목록이다.
> 가이드 콘텐츠(`app/guide/_components/guide-data-*.ts`) 작성 시 이 문서의 기능이 빠짐없이 반영되어야 한다.

---

## 1. 강사(instructor) 권한

### 1.1 홈 / 로그인 (`/`, `/login` — `app/page.tsx`, `app/login/page.tsx`)
- 로그인 다이얼로그 열기 — 아이디/비밀번호로 입장
- 회원가입 폼 전환 — 새 계정 만들기 (역할 선택 포함)
- 사용 설명서 링크 — 가이드 페이지 접근

### 1.2 강좌 목록 (`/programs` — `app/programs/page.tsx`)
- 새 강좌 만들기 버튼 → 생성 다이얼로그
  - 제목(필수), 설명(선택), 시작일/종료일(선택)
  - 매주 수업 요일 체크박스(월~일 복수 선택, 1주차 실제 수업 날짜 미리보기)
  - 강좌 생성 → 상세 페이지 이동
- 강좌 카드: 제목·설명·운영자, 주차수/강의수/수강생수 배지, 기간·요일, 마지막 수정일
- 강좌 보기 버튼 — 상세 페이지 이동
- 강좌 정보 수정 버튼(아이콘) — 제목/설명/기간/요일 수정 다이얼로그
- 강좌 삭제 버튼(아이콘) — 확인 후 제거

### 1.3 강좌 상세 (`/programs/[id]` — `app/programs/[id]/page.tsx`)
**상단 정보**
- 강좌 제목·설명·배지(주차/강의/수강생/기간/요일) 표시
- 강좌 정보 수정 버튼
- 초대 링크 복사 버튼 — 초대 코드와 참여 URL 복사

**진행률 대시보드(강사 전용)**
- 완료 수강생 수 / 전체 수강생 수, 평균 진행률(%)

**주차별 강의 탭**
- 주차 빠른 이동 칩(1주차, 2주차…) — 해당 주차로 점프
- 주차 추가 버튼 — 다음 번호 주차 생성
- 주차 수정 메뉴: 이름 변경, 주차 삭제(확인 후 하위 항목 포함 삭제)
- 항목 추가 버튼 6종 (주차마다):
  - 강의 추가 — 제목 입력
  - 공지 추가 — 제목 입력
  - 게시물 추가 — 제목 입력
  - 시험 추가 — 제목 + LLM 선택(ChatGPT/Claude/Gemini/Grok) + 미션 입력
  - 연습 추가 — 제목 + LLM 선택
  - 실습(아뜰리에) 추가 — 제목 + 장르(시/동화책/에세이/빈 작품) + 템플릿 + 안내문
- 항목 카드: 타입별 아이콘·색, 제목, 페이지수/LLM 로고/장르 배지, 진행률(X명 완료, 평균 Z%)
- 드래그 핸들 — 항목 순서 정렬
- 항목 메뉴(...): 편집(강의/공지/게시물), 삭제(확인 후 원본 포함 제거)

**수강생 명단 탭(강사 전용)**
- 수강생 등록 버튼: 아이디 검색(디바운스) → 체크박스 복수 선택 → 칩 표시 → 등록
- 초대 코드 안내·복사
- 명단 행: 아바타, 이름/@아이디, 등록일, 진도 바(X/Y 강의, %)
- 행 클릭 — 학생별 크레딧 대시보드(`/programs/[id]/students/[userId]`)
- 내보내기 버튼 — 확인 후 강좌에서 제거

### 1.4 강의 빌더 (`/build/[id]` — `app/build/[id]/page.tsx`)
- 상단: 뒤로가기, 강의 제목, 저장 상태 표시(저장됨/저장 중/변경 있음), 저장 버튼
- 페이지 네비: 페이지 목록 클릭 전환, 이름 변경, 순서 이동, 삭제(최소 1페이지), 새 페이지 추가
- 블럭 팔레트: 텍스트/이미지/비디오/코드/인용/구분선/AI 초안 등 추가
- 블럭 편집: 인라인 콘텐츠 수정, 드래그 정렬(삽입선 표시), 복제, 위/아래 이동, 삭제
- AI 콘텐츠 블럭: 프롬프트 입력 → 생성 실행(이미지·영상·음성·글) — 데모에선 프롬프트만 세팅, 실행은 사용자가 직접
- 우측 도구: 본문 최대 폭(narrow/normal/wide/full), 미리보기(학생 뷰) 전환, JSON 다운로드/업로드(백업·복원)

### 1.5 공지 상세·편집 (`/announcements/[id]`, `/announcements/[id]/edit`)
- 상세: 뒤로가기, 목차 사이드바(클릭 스크롤), 블럭 문서 뷰, 편집 버튼(강사만)
- 편집: 제목 입력, 블럭 캔버스 편집, 저장

### 1.6 게시물 상세·편집 (`/posts/[id]`, `/posts/[id]/edit`)
- 상세: 공지와 동일 + 댓글 섹션
- 댓글: 목록 보기, 입력·제출, 삭제(본인 것, 강사는 타인 것도)
- 편집: 제목 입력, 블럭 캔버스 편집, 저장

### 1.7 시험·연습 드릴 (`/drill/[id]` — `app/drill/[id]/page.tsx`)
- 선택한 LLM 클론 전체 화면 표시 (ChatGPT/Claude/Gemini/Grok)
- 강사는 설정한 미션·드릴 확인 가능
- 나가기 — 강좌 상세로 복귀

### 1.8 창작 실습 아뜰리에 (`/atelier/[id]` — `app/atelier/[id]/_components/atelier-editor.tsx`)
- 시범본(sample) 에디터 — 학생 참고용 시범 작품 편집
- 장르별 블럭(시/동화책/에세이): 추가, 편집, 삭제
- 저장, 미리보기 토글, PDF 다운로드(책 형태)

### 1.9 AI 스튜디오 (`/studio` — `app/studio/page.tsx`)
- AI 생성 사용량 대시보드 — 누적 횟수·비용
- 생성 로그 — 기능별 상세 기록

### 1.10 내 정보·설정 (`/settings` — `app/settings/page.tsx`)
**내 정보 탭**
- 프로필: 아바타, 닉네임, 역할 배지, @아이디 표시
- 프로필 사진 선택 — 카테고리별(동물/꽃/자연) 아바타 그리드, 선택 즉시 저장

**설정 탭**
- 강좌 콘텐츠 카드 모양(4종): 솔리드 타일 / 모노 미니멀 / 커버 / 콤팩트 (미리보기 포함)
- 콘텐츠별 묶기 ON/OFF — 주차 안 항목을 종류별 그룹화
- 글자 크기(3단계): 보통/크게/아주 크게
- 화면 크기(3단계): 버튼·여백·아이콘 확대
- 모서리 둥글기(5단계): 각짐~많이
- 고대비 모드 토글
- 어두운 화면(다크 모드) 토글
- 로그아웃 버튼

### 1.11 상단바 (`components/site-header.tsx`)
- 로고 클릭 — 홈 이동
- 사용자 드롭다운: 사용 설명서(/guide), 내 정보·설정(/settings), 내 강좌(/programs), AI 스튜디오·사용량(/studio), 로그아웃

## 2. 학생(student) 권한

### 2.1 홈 / 로그인 / 참여 (`/`, `/login`, `/join/[code]`)
- 로그인 다이얼로그 — 아이디/비밀번호 입장
- 회원가입 폼 전환
- 초대 링크 클릭 — 강좌 자동 참여 (비로그인 시 로그인 후 복귀)
- 참여 완료 후 강좌 상세로 자동 이동

### 2.2 강좌 목록 (`/programs`)
- 초대 코드 참여 버튼 — 코드 입력 다이얼로그 → 강좌 참여
- 강좌 카드 클릭 — 상세 이동

### 2.3 강좌 상세 (`/programs/[id]`)
- 주차 빠른 이동 칩
- 항목별 큰 액션 버튼: 강의 "학습하기"(→/learn), 공지·게시물 "보기", 실습 "만들기"(→/atelier), 시험 "시험보기"/연습 "연습하기"(→/drill)
- 내 진행률 표시

### 2.4 학습 플레이어 (`/learn/[id]` — `app/learn/[id]/_components/player.tsx`)
**페이지 탐색**
- 이전/다음 버튼, 페이지 점(dot) 클릭 이동, 키보드 ←/→, 강좌로 돌아가기

**블럭 타입별 학습 행동**
- 텍스트/제목/인용/강조 박스/표/차트/북마크 — 읽기·확인
- 동영상(업로드): 재생/일시정지, 시간 조절, 풀스크린
- 동영상(생성)·음성(생성)·이미지(생성)·AI 글쓰기(LLM): 프롬프트 입력·수정 → "생성하기/실행하기" 버튼 → 생성 시간·비용(크레딧/원화) 표시 → 히스토리 버튼으로 이전 결과 열람·복원
- 음성(업로드): 재생/일시정지, 파형 드래그
- 이미지(업로드): 보기

**학습자 자랑방(채팅 패널)**
- 메시지 입력·전송
- 첨부(📎): 내 AI 작품 선택 또는 파일 업로드
- 메시지 우클릭 → 이모지 반응(👍❤️😂😮😢👏🎉🔥)
- 채팅 속 블럭 링크 클릭 — 해당 블럭으로 이동
- 블럭 우클릭 → "채팅에 링크"
- 패널 토글(데스크톱) / 전체 화면 열기·닫기(모바일)

### 2.5 시험·연습 드릴 (`/drill/[id]`)
- LLM 클론 UI(ChatGPT/Claude/Gemini/Grok)에서 질문 입력·전송, 스트리밍 답변, 대화 계속
- 시험: 미션 확인 → AI 답변 1회 이상 시 제출 버튼 활성화 → 제출 → 축하 화면 → 다시 해보기 / 강좌로 돌아가기
- 연습: 자유 대화, 언제든 나가기

### 2.6 창작 실습 아뜰리에 (`/atelier/[id]`)
- 강사 시범본 보기
- 표지 편집: 제목/부제/저자명, 표지 그림 AI 생성 또는 업로드, 테마/글꼴/비율 선택
- 페이지(쪽) 관리: 추가, 제목 편집, 순서 이동, 삭제
- 본문 블럭: 텍스트/제목/인용/강조/이미지(AI·업로드)/구분선/표 추가·편집·이동·복제·삭제
- 미리보기 패널 — 완성 모습 확인
- 저장, PDF 내려받기

### 2.7 게시물 (`/posts/[id]`)
- 본문·목차 보기, 목차 클릭 스크롤
- 댓글 작성·제출, 비밀 댓글 토글, 대댓글, 첨부(📎), 본인 댓글 삭제

### 2.8 공지 (`/announcements/[id]`)
- 본문·목차 보기

### 2.9 AI 스튜디오 (`/studio`)
- 내 생성 사용량·비용 대시보드, 생성 로그

### 2.10 내 정보·설정 (`/settings`)
- 강사와 동일 (역할 배지만 "학생"): 아바타 변경, 카드 모양, 묶기, 글자/화면 크기, 둥글기, 고대비, 다크 모드, 로그아웃

### 2.11 상단바
- 강사와 동일 메뉴 (사용 설명서 / 내 정보·설정 / 내 강좌 / AI 스튜디오·사용량 / 로그아웃)

## 3. 공통 UX 패턴 (가이드에 녹일 것)

- 토스트 알림: 성공(초록)/실패(빨강)
- 삭제류 동작은 항상 확인 다이얼로그
- 진행률: 강좌 카드·상세·플레이어 전반에 표시
- AI 생성: 프롬프트 → 실행 → 시간·비용 표시 → 히스토리 복원 (데모 정책: 결과물은 비워두고 실행은 사용자가 직접)
- 접근성: 글자 3단계, 화면 3단계, 고대비, 다크 모드, 둥글기 5단계 — `data-*` 속성으로 전역 적용
