> Instructorly 프론트엔드 프로토타입(Next.js 16 · App Router)의 **소스코드를 역설계**해 정리한 화면 지도입니다. 총 **14개 화면(라우트)** · **2개 역할**(강사 · 수강생, 일부 공용). 본문 **Pretendard** / 라벨 Geist Mono, 모노크롬 디자인, 노인 친화 접근성(글자 크기 3단계 · 고대비 · 다크 테마).
>
> 역할 표기 — **공용**: 누구나 · **강사**: instructor 전용 · **수강생**: student 전용

## 1. 화면 메뉴 구조도

![화면 메뉴 구조도](IMG_MENU)

### 화면별 설명 (핵심 요약)

위 구조도의 5개 영역 순서대로 정리했습니다.

**진입 · 인증**
- **랜딩 `/`** — *공용*. "누구나 AI 강사가 됩니다" 서비스 소개 + 시작 CTA. 정적 페이지로, 로그인·강좌 둘러보기로 유도.
- **로그인 · 회원가입 `/login`** — *공용*. 아이디·비밀번호 + 역할(강사/학생) 선택(프로토타입용 평문 인증). 성공 시 `/programs`로 이동. 데모 계정 `teacher/1234`, `kim/1234`.

**강좌 운영 (프로그램)**
- **강좌 목록 `/programs`** — *공용(역할 분기)*. 로그인 후 **기본 진입점**. 내 강좌 카드 그리드. 강사=「새 강좌 만들기」, 학생=「초대 코드로 참여」.
- **강좌 상세 `/programs/[id]`** — *공용(역할 분기)*. **주차(Week)별 혼합 콘텐츠 허브** — 강의·게시물·공지·시험·연습을 한 주차에 순서대로 배치. 강사=주차/항목/수강생 운영 + 초대 링크, 학생=학습 링크 + 내 진행률. 상단 진행률 대시보드.
- **수강생 작업 · 진행 `/programs/[id]/lessons/[lessonId]/students`** — *강사 전용*. 특정 강의의 학생별 진도 + AI 블럭 작업물·생성 이력을 마스터-디테일로 열람(검색·정렬).
- **초대 링크 참여 `/join/[code]`** — *수강생*. 초대 코드가 박힌 링크로 들어오면 자동 등록 후 강좌 상세로 이동(비로그인 시 로그인 경유 복귀).

**강의 (Course)**
- **강의 목록 `/courses`** — *공용*. 전체 강의 카드. 「학습하기」(누구나), 강사=「새 강의 만들기」·편집·삭제.
- **블럭 빌더 `/build/[id]`** — *강사*. 노션식 블럭 에디터 — 팔레트 **드래그** 또는 **`/` 슬래시**로 9종 블럭(제목·텍스트·인용·콜아웃·구분선·이미지·동영상·오디오·LLM) 삽입. 이미지·LLM·미디어는 **AI 생성 + 소요시간 표시**. 페이지 단위 구성, 저장(⌘/Ctrl+S).
- **학습 플레이어 `/learn/[id]`** — *수강생*. **스크롤이 아닌 페이지 넘김** 학습. 이미지·LLM 블럭을 본인 세션에서 다시 생성(이력 보존), 진도 자동 기록. 오른쪽에 **자랑방 채팅**(4초 폴링) 패널.

**커뮤니티**
- **게시물 `/posts/[id]`** — *공용 보기 / 강사 편집*. 블럭 본문 + **댓글·대댓글·비밀 댓글**(5초 폴링). 프로그램 주차 항목에서 진입.
- **게시물 편집 `/posts/[id]/edit`** — *강사*. 제목 + 블럭 에디터, 저장(⌘/Ctrl+S). 새 글도 먼저 생성된 뒤 이 화면에서 내용을 채움.
- **공지사항 `/announcements/[id]`** — *공용 보기 / 강사 편집*. 블럭 본문(**댓글 없음** — 읽기 전용 공지).
- **공지 편집 `/announcements/[id]/edit`** — *강사*. 게시물 편집과 동일 구조.

**평가 · 연습**
- **시험 · 연습 `/drill/[id]`** — *수강생(강사 미리보기)*. 실제 LLM 사이트(**ChatGPT · Claude · Gemini · Grok**) UI를 똑같이 흉내 낸 클론 화면에서 프롬프트 실습. **시험(exam)**=미션 + 제출, **연습(practice)**=자유. 외부 API 없이 클라이언트 가짜 스트리밍.

## 2. 유저 플로우 — 강사

![강사 유저 플로우](IMG_INSTRUCTOR)

1. **로그인** — "강사" 역할로 시작 → `/programs`
2. **강좌 만들기** — 제목·기간·요일 입력, 초대 코드 자동 발급 → `/programs/[id]`
3. **주차 추가** — 운영 리듬에 맞춰 주차 구성
4. **콘텐츠 제작·배치** — 강의(블럭 빌더 + AI 생성) / 게시물·공지 / 시험·연습(LLM 선택)을 주차에 추가
5. **수강생 모으기** — 아이디 검색 등록 또는 초대 링크 공유, 드래그로 항목 순서 배치
6. **현황 확인** — 진행률 대시보드 + 강의별 수강생 작업/진도 열람

## 3. 유저 플로우 — 수강생

![수강생 유저 플로우](IMG_STUDENT)

1. **로그인** — "학생" 역할로 시작 → `/programs`
2. **강좌 참여** — 초대 코드 입력 또는 초대 링크(`/join/[code]`)로 자동 등록
3. **강좌 상세** — 주차별 항목 + 내 진행률 확인
4. **학습하기** — 강의(페이지 넘김 + AI 블럭 재생성) / 게시물·공지 보기·댓글 / 시험·연습(AI 클론 실습)
5. **자랑·누적** — 자랑방 채팅에 결과물 공유(4초 폴링), 진도 자동 기록
